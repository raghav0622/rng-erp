- **BaseEntity**: The canonical base for all persisted entities, defined in rng-repository.
- **Feature**: A unit of business capability, registered at kernel bootstrap.
- **RBAC**: Role-Based Access Control, deterministic and explainable.
- **Assignment**: A mapping of user, resource, and role.
- **AuditEvent**: A canonical event for platform-level logging.
- **Invariant**: A rule that must always hold true for a domain or contract.
- **Kernel**: The core execution and contract layer of the platform.
- **Repository**: The frozen, client-safe data access layer.
- **Contract**: A TypeScript interface or type that defines the public surface and invariants for a domain.
